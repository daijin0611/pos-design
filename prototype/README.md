# POS H5 高保真原型 — 丽德健康管理系统

> 健康美业管理平台移动端原型，覆盖 **会员端 / 员工端 / 管理端** 三个角色，共计 18 个功能页面。

## 快速开始

```bash
# 进入原型目录
cd prototype

# 安装依赖
npm install

# 启动服务器
npm start

# 访问
open http://localhost:3000
```

## 技术栈

| 项目 | 说明 |
|------|------|
| 纯 HTML/CSS/JS | 无框架依赖，静态页面原型 |
| Express | 独立开发服务器（端口 3000） |
| 设计系统 | 自定义 CSS 变量体系 |

正式开发技术栈：**Vue3 + Vant 4**，运行于微信公众号 H5。

## 设计规范

### 色彩体系

| 角色 | 主色 | 色值 | 用途 |
|------|------|------|------|
| 品牌 | 薄荷绿 | `#2CB5A0` | 主按钮、会员端主题 |
| — | 玫瑰粉 | `#F5A0B5` | 辅助色、女性化场景 |
| — | 琥珀金 | `#E8A849` | 强调色、警告、评分 |
| 员工 | 靛蓝 | `#4A6CF7` | 员工端主题色 |
| 管理 | 深蓝 | `#1A1A2E` | 管理端主题色、桌面背景 |

### 布局参数

- **设备基准**：iPhone 375 × 812
- **移动端**：全屏自适应（100vh / 100dvh），safe-area 适配刘海/底部
- **桌面端**（>430px）：居中手机框架，圆角 24px + 投影
- **字体**：系统默认 `-apple-system, PingFang SC, Microsoft YaHei`
- **圆角**：卡片 14px / 按钮 pill / 小组件 10px

### 响应式策略

```

│              │      │      深蓝背景 #1A1A2E      │
│  全屏页面     │      │  ┌──────────────────┐    │
│  无模拟状态栏 │      │  │  375px 手机框架   │    │
│  真实安全区   │      │  │  圆角 + 投影      │    │
```

## 页面总览

### 🌐 全局（1 页）

| 页面 | 文件 | 说明 |
|------|------|------|
| 登录/授权 | `global-login.html` | 手机验证码 + 微信登录 + 角色选择（会员/员工/管理） |

### 👤 会员端（7 页）

| 页面 | 文件 | 功能 |
|------|------|------|
| 会员首页 | `member-home.html` | 余额卡片、快捷操作（7个入口）、最近预约、消费记录 |
| 余额查询 | `member-balance.html` | 余额/累计充值/消费、交易明细列表、按月筛选 |
| 预约服务 | `member-appointment.html` | 选门店→选项目→选时间→选技师→确认预约 |
| 优惠券 | `member-coupon.html` | 可使用/已使用/已过期分类、优惠券详情 |
| 门店查询 | `member-store.html` | 门店列表+地图定位+详情+导航 |
| 会员二维码 | `member-qrcode.html` | 动态二维码展示+会员信息卡 |
| 积分 | `member-points.html` | 积分余额/明细+积分兑换商城 |

### 👨‍💼 员工端（9 页）

| 页面 | 文件 | 功能 |
|------|------|------|
| 员工首页 | `staff-home.html` | 今日统计、工资速览、快捷入口（8个）、待跟进列表 |
| 预约管理 | `staff-appointment.html` | 日期切换、状态筛选（待确认/已确认/已完成）、确认/拒绝/开始服务 |
| 工资查询 | `staff-salary.html` | 月份切换、总工资+3类分项、业务明细+筛选（项目/产品/疗程券/充值） |
| 回访/生日提醒 | `staff-reminder.html` | 双 Tab 面板：回访提醒（5条/紧急程度）+ 生日提醒（3条/日期分组） |
| 会员修复调理记录 | `staff-member-record.html` | 搜索选会员→选服务项目→回访安排→描述详情→拍照上传 |
| 满意度 | `staff-satisfaction.html` | 综合评分、评分分布、趋势图、服务项目排行、最新评价列表 |
| 手机收银 | `staff-checkout.html` | 选会员→选项目/产品→购物车→优惠券→结算 |

### 📊 管理端（3 页）

| 页面 | 文件 | 功能 |
|------|------|------|
| 管理首页 | `admin-home.html` | 数据看板（营业额/订单/客流/满意度）、业绩趋势图、员工排名 |
| 业绩查询 | `admin-performance.html` | 门店/员工维度业绩数据、分类统计、图表 |
| 排名 | `admin-ranking.html` | 项目排名+员工业绩排名+工资排名、时间筛选 |

## 交互说明

原型已内置以下交互效果，可在浏览器中直接体验：

| 交互 | 说明 |
|------|------|
| Tab 切换 | 顶部 Tab 点击切换内容面板（提醒页、预约页、满意度页） |
| 筛选切换 | 工资明细筛选（全部/项目/产品/疗程券/充值）、预约状态筛选 |
| 日期切换 | 工资月份切换、预约日期选择 |
| 搜索交互 | 会员修复调理记录：搜索框聚焦→下拉结果→选中填入 |
| 服务选择 | 单选卡片点击高亮 |
| 动画效果 | 页面元素渐入动画（fadeInUp + delay） |
| 滚动体验 | 卡片列表流畅滚动、桌面端 sticky Tab 栏 |

## 浏览器预览

### 桌面端

直接打开 `http://localhost:3000`，页面以手机框架居中展示，右上角有三端快速导航。

### 移动端真机

1. 确保手机和电脑在同一局域网
2. 查看电脑 IP（如 `192.168.1.100`）
3. 手机浏览器访问 `http://192.168.1.100:3000`
4. 页面自动全屏适配，无模拟状态栏

### Chrome DevTools 模拟

1. F12 打开开发者工具
2. 切换设备模拟（iPhone X / 375×812）
3. 刷新页面即可
