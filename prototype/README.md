# POS H5 高保真原型 — 丽德健康管理系统

> 健康美业管理平台移动端原型，覆盖 **会员端 / 员工端 / 管理端** 三个角色。
> **当前主页仅展示二期范围功能**（M0-M3 / E0-E2 / A0-A3 / P5），二期外页面文件保留但已从入口隐藏。

## 快速开始

```bash
# 进入原型目录
cd prototype

# 安装依赖
npm install

# 启动服务器
npm start

# 访问
open http://localhost:3000
```

## 技术栈

| 项目 | 说明 |
|------|------|
| 纯 HTML/CSS/JS | 无框架依赖，静态页面原型 |
| Express | 独立开发服务器（端口 3000） |
| 设计系统 | 自定义 CSS 变量体系 |

正式开发技术栈：**Vue3 + Vant 4**，运行于微信公众号 H5。

## 设计规范

### 色彩体系

| 角色 | 主色 | 色值 | 用途 |
|------|------|------|------|
| 品牌 | 薄荷绿 | `#2CB5A0` | 主按钮、会员端主题 |
| — | 玫瑰粉 | `#F5A0B5` | 辅助色、女性化场景 |
| — | 琥珀金 | `#E8A849` | 强调色、警告、评分 |
| 员工 | 靛蓝 | `#4A6CF7` | 员工端主题色 |
| 管理 | 深蓝 | `#1A1A2E` | 管理端主题色、桌面背景 |

### 布局参数

- **设备基准**：iPhone 375 × 812
- **移动端**：全屏自适应（100vh / 100dvh），safe-area 适配刘海/底部
- **桌面端**（>430px）：居中手机框架，圆角 24px + 投影
- **字体**：系统默认 `-apple-system, PingFang SC, Microsoft YaHei`
- **圆角**：卡片 14px / 按钮 pill / 小组件 10px

### 响应式策略

```

│              │      │      深蓝背景 #1A1A2E      │
│  全屏页面     │      │  ┌──────────────────┐    │
│  无模拟状态栏 │      │  │  375px 手机框架   │    │
```

## 交互说明

原型已内置以下交互效果，可在浏览器中直接体验：

| 交互 | 说明 |
|------|------|
| Tab 切换 | 顶部 Tab 点击切换内容面板（提醒页、预约页、满意度页） |
| 筛选切换 | 工资明细筛选（全部/项目/产品/疗程券/充值）、预约状态筛选 |
| 日期切换 | 工资月份切换、预约日期选择 |

| 动画效果 | 页面元素渐入动画（fadeInUp + delay） |
| 滚动体验 | 卡片列表流畅滚动、桌面端 sticky Tab 栏 |

## 浏览器预览

### 桌面端

直接打开 `http://localhost:3000`，页面以手机框架居中展示，右上角有三端快速导航。

### 移动端真机

1. 确保手机和电脑在同一局域网
2. 查看电脑 IP（如 `192.168.1.100`）
3. 手机浏览器访问 `http://192.168.1.100:3000`
4. 页面自动全屏适配，无模拟状态栏

### Chrome DevTools 模拟

1. F12 打开开发者工具
2. 切换设备模拟（iPhone X / 375×812）
3. 刷新页面即可
